# 📊 InsightFlow AI Dashboard

An AI-powered data analytics dashboard that lets you upload any CSV dataset and instantly get interactive visualizations and AI-generated insights using Google Gemini.

## ✨ Features
- 📁 Upload any CSV dataset
- 🤖 AI insights using Google Gemini 2.0 Flash
- 📊 Interactive charts — bar, pie, line
- 🌍 Filter by region, category, date range
- 📋 Full data table with search & download
- 🎨 Dark theme professional UI
- 📱 Mobile friendly & responsive

## 📦 Installation

```bash
# Clone the repo
git clone https://github.com/pudileena20-ops/hackthon_InsightFlowAiDashboard.git

# Go to project folder
cd hackthon_InsightFlowAiDashboard

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

---

## 🔑 API Key Setup

Create `.streamlit/secrets.toml` file:
```toml
GEMINI_API_KEY = "your_gemini_api_key_here"
```

Get your free API key at: [aistudio.google.com](https://aistudio.google.com)
